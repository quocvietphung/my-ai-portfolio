## ğŸ”‘ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/quocvietphung/my-ai-portfolio.git
   cd my-ai-portfolio
   ```
2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```
3. **Set up environment variables**

   Create a `.env.local` file:
   ```env
   AZURE_OPENAI_API_KEY=your_api_key

   ```
4. **Run locally**
   ```bash
   npm run dev
   ```
